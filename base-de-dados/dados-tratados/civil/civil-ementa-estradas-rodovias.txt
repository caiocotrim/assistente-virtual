OBRIGATÓRIO NÚCLEO: Específico 
PRÉ-REQUISITO: Topografia. 
EMENTA: Classificação das rodovias e o conceito de nível de serviço; Escolha do traçado 
de uma estrada; Elementos básicos para o projeto geométrico de rodovias; Características 
técnicas para projeto; Curvas horizontais circulares; Curvas horizontais de transição; 
Superelevação; Superlargura; Estudo do perfil longitudinal e o projeto de 
curvas de concordância vertical.
