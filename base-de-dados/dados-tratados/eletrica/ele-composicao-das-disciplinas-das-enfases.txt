Composição das Disciplinas das Ênfases 
Para cada ênfase oferecida no curso está previsto um conjunto de componentes 
curriculares  próprias.  Para  cada  ênfase  estão  previstas  10  componentes  curriculares, 
conforme as Tabela 16 a Tabela 18. Todas as componentes de ênfase se enquadram no 
Núcleo de Componentes Específicos. 
A oferta das componentes curriculares das ênfases e das optativas seguirão as 
estratégias acadêmicas definidas pela coordenação do curso. 
As componentes curriculares referentes à ênfase de Automação e Controle estão 
descritas na Tabela 16. As componentes curriculares referentes à ênfase de Eletrônica 
estão  descritas  na  Tabela  17.  As  componentes  curriculares  referentes  à  ênfase  de 
Eletrotécnica estão descritas na Tabela 18.  
A caracterização de cada ênfase se dá por meio de um grupo de componentes 
curriculares que compõem os critérios relacionados na Resolução n° 1010 de 2005 do 
CONFEA.  Entretanto,  na  matriz  curricular,  as  três  ênfases  são  contempladas  por 
componentes curriculares obrigatórias e componentes curriculares específicas dedicadas 
a ênfase. Essa estratégia permite ao discente explorar a flexibilidade curricular de forma 
que, uma vez cursadas as componentes mínimas, seja possível a habilitação em até duas 
ênfases. 
Neste projeto pedagógico, a caracterização de uma ênfase perpassa pelo grupo de 
componentes curriculares dedicados à cada modalidade, considerando um total de 360 
horas. Sendo assim, para que o egresso consiga obter a habilitação em duas ênfases, será 
obrigatório cursar as seis componentes de ênfases, contabilizando 360 horas, mais as duas 
optativas, contabilizando mais 120 horas, totalizando, então, 480 horas, sendo 240 horas 
para cada ênfase escolhida. A totalização de 360 horas para cada ênfase se efetiva por 
meio de componentes obrigatórias na matriz curricular que compõem o restante de 120 
horas.
